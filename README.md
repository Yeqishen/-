## 问题：当你是动物饲养员，5只猴子只有四根香蕉分时，应该怎么办?

我先告诉所有的猴子一个坏消息——今天免费的香蕉一支都没有了。吞一下口水，再告诉沮丧的猴子们一个好消息——今天收费的香蕉还剩2支。
	啥？收费？没错——每支香蕉收费1.5香蕉，至于多出来那半支，等你香蕉多了，再还。此时猴子们纷纷觉得自己被耍了，拒绝购买天价香蕉。就这么耗到了晚上，猴子们都饥肠辘辘，终于有一只嘴馋的猴子忍不住了，贷款买了一支香蕉，大口吃了起来，为了方便，我们就叫它猴甲。其他猴子闻着香蕉味，肚子叫的慌，每只猴子都纷纷表示要买剩下那支香蕉。我摇摇头，只剩1支蕉了，香蕉本来实行先到先得制度，既然你们都想买，那只能摇号了。此时，猴甲抹了抹嘴，拿起它吃剩地半支香蕉对其他猴子说：我这里还有半支蕉，你们要赌自己摇号的运气呢，还是打算先吃半支蕉垫垫肚子？猴乙摸着自己的肚子问了一句：那你这半支蕉，打算卖多少钱？猴甲笑了笑：卖1支蕉的价！众猴皆惊，你这利息居然达到了100%！猴甲笑了笑，那没办法咯，待会摇不到饲养员那支蕉，你们就等着饿肚子吧。此时，猴乙仔细转了一下脑袋瓜子，以一只蕉的代价，从猴甲手里买过了那半支蕉。残酷的摇号从没有得吃蕉的猴丙、猴丁、猴戊中进行。幸运的是，猴丙摇中了。记账之后，猴丙悄悄对猴丁、猴戊说：这支蕉，我必须吃一半，剩下的一半，卖价跟猴甲一样，你们谁要不想饿肚子，就赶紧来。
	听到这，我越发觉得有趣，就对猴子们说，我手里还有两支本来属于明天配额的香蕉，现在可以让你们提前预支，我要说明的是，明天也没有免费的香蕉。猴子们一听，大惊。还没得吃过蕉的猴丁不知哪来的勇气，马上表示：两支蕉我全要了！好了，现在把猴子们的负债和持有情况整理一下：猴甲：负债1.5蕉，持有1蕉（期权）猴乙：负债1蕉（期权），持有0蕉猴丙：负债1.5蕉（期权），持有1蕉（现货）猴丁：负债3蕉，持有2蕉（现货）猴戊：负债0蕉，持有0蕉此时的猴丁，独占两蕉，意气风发。其他没有蕉的猴子们慌了。
	欠蕉的不知道怎么办，因为不能凭空生出蕉来；没得吃蕉的猴虽然持有，却也不敢乱吃蕉了。此时，一直没买蕉的猴戊发话了：我一直不买蕉，是因为我表弟明天一早就要来看我了，表弟是一只专门种香蕉的猴，它手里有大把香蕉，到时，你们手里的蕉就不值钱了。
  此话一出，仿佛晴天霹雳，猴丁差点没崩溃。猴戊接着说：但是，不吃蕉，我连今晚都顶不过去，所以猴丙，你手里那半支蕉我可以买下来，价格当然不可能像你喊的那样，我只愿出半支蕉。猴丙想了想：现在猴甲、猴乙都身背债务，会买蕉的猴就只有戊了，赶紧答应。而此时，猴丁更紧张，他表示，手里的蕉也可以按原价卖给猴戊。于是，猴乙发话了，干脆这样吧，猴丁你手里那两支蕉，你自己吃半支，剩下的1.5支蕉以1蕉的价格卖我。猴丁愣了一下，当我大傻子吗，1.5蕉换1蕉，你不是空手套白狼吗？猴乙笑了笑说，你要不愿意，今晚香蕉一烂，明天你就得哭了，你看他们几个谁还愿意买你的蕉。猴丁只能照做。此时我们再统计一下各猴子手里的香蕉甲：持有0（吃了0.5），负债0.5乙：持有1.5（吃了0.5），负债2丙：持有0（吃了0.5），负债1丁：持有0（吃了0.5），负债2戊：持有0（吃了0.5），负债0.5谁知，猴乙从猴丁处拿到一支半的香蕉后，立马丢了半支给猴戊。
  猴子们这时才明白，猴戊根本没有什么种香蕉的表弟。这一切都是猴乙和猴戊为了做空香蕉的价格而上演的一出闹剧。猴丁恼羞成怒，变身大金刚，一拳砸到了猴乙身上，打算把香蕉抢回来，猴乙好歹是猴子，灵活地闪躲着猴丁的攻击。俩猴打得正酣，幸被其他猴子拉开了。
  猴戊把手里的半根香蕉给猴丁，并告诉他：你身手不错，我愿意用半支香蕉雇佣你做我的保镖和助手。猴丁本来也迁怒于猴戊，但听他这么一说，空有一身力气也是白费，还不如减轻一点债务，遂点头答应了。这时，一直没讲话的猴甲发话了：香蕉这东西，其实远远没有你们说的那样放不久，只要它还是完整的，就能放好几天呢，但要是只剩下半支，那就不好说咯。说完，若有所思地看了一眼猴丁。猴丁气急败坏，转过头来又想去打猴甲。见到事情愈演愈烈，我赶紧介入了猴子们的斗争。我提出：你们手上所持有但现在不想吃的香蕉，统统可以交给我，我这边帮你们放进冰箱里冻着，你们什么时候想吃就来找我要，对了，当你存够10支香蕉，每两天就免费多发给你半支香蕉。猴丁和猴乙乖乖地把手里的香蕉交了出来。下面我们再来总结一下猴子们现在的欠款情况猴甲：负债0.5支猴乙：负债1支，储蓄1支猴丙：负债1支猴丁：负债2支，存款0.5支猴戊：负债0.5支猴子们都吃了半只香蕉，也填饱了肚子，看着夜色还美，就闲聊起来。猴甲感慨到：咱今天各只吃了半只香蕉，却都背负着债务，明天咱是不是得喝西北风了。猴丁流着泪说：都怪你们，香蕉本来就是用来吃的，你们却拿来炒，哼！猴戊想了想，说：兄弟们，如果我有办法让咱大赚一笔，后面几天都有吃不完的蕉，你们愿不愿意跟我做？四只猴子立马来了精神：当然啦，有啥输不起的。猴戊忽然转头对我说：饲养员小哥哥，你觉得我这只猴子，值几支香蕉？我愣了一下：你想干啥？猴戊：没干啥，就是问问那，想看看我身价几何。我想了下，随口说了句：你起码值100支香蕉吧？猴戊两眼放光：那好啊，我现在把自己抵押给你，你发给我100支香蕉呗？我愣了一下：你没审题吗？5只猴子只有4根香蕉，我上哪里找100支香蕉给你？猴戊笑了笑：要不这样，你写十张条子给我，大概意思就是“凭此条子可以找饲养员兑换10支香蕉”给我就行啦。我不解：那你把自己抵押给我，如果你还不起香蕉，我咋办？猴戊说：还不起你就把我送去动物表演中心打工呗，那边肯定出得起100支香蕉。我想了一下，也是，反正都是动物园，动物表演中心正缺猴表演跳水呢，我把它送过去，别说100支香蕉了，1000支他们也会给，不亏！但我也没这么笨，我说道：“抵押归抵押，利息还是要算的，抵押时间算一星期吧，一星期后，你得还给我110支香蕉，可以吗？”猴戊答应了，我立马开了条。猴戊拿着条子，马上跟其他四只猴说，你们赶紧照做。于是，五只猴子开出了五十张香蕉卖身契。猴戊笑嘻嘻地跟我说：饲养员小哥哥，我有个表弟，就关在对面猴山里，他们那边猴多，我想带着我舍友过去跟他们联谊一趟，你就准一下呗。我想了想，整天把猴关在这儿也挺闷，就答应了，但告知他们，不能打架，否则增加债务，猴儿们嬉皮笑脸地答应了。
  五只猴子来到了对门的猴山，受到猴山一众猴子们热烈的欢迎。这间猴舍的的饲养员是我的广西舍友，他喜欢用栗子喂猴子，因为栗子个小，又好储存，所以这边的猴子们多多少少都有点积蓄。我悄悄地跟了过去，想看看五只猴们到底在谋划着干啥。寒暄之后，五只猴儿就散开在了猴山的各个角落。我暗暗跟着猴戊，想看看他打算怎么做。只见猴戊四处打量了一下他表弟的猴窝，两眼热泪盈眶“表弟啊，你平时就睡这玩意儿”把他表弟弄的一愣一愣的。猴戊接着说“你们平时吃啥呢？难不成连香蕉都吃不着？”表弟说“咱这儿都吃栗子，栗子也挺好吃啊”猴戊一听，眼泪就掉下来了，接着它开始长篇论述香蕉是如何如何美味、入口即化、香飘十里，简直是猴间极品，说得表弟那边一群猴潸然泪下——我们饲养员是不是虐待我们……良久，猴戊擦了擦眼泪，告诉表弟“你哥我平时省吃俭用，攒下了100支香蕉，都冻在饲养员的冰柜里呢，你们就拿点栗子过来跟我交换，也算给你们开开荤了”猴儿们一听，还有这种好事儿，纷纷拿出积攒已久的栗子。这边的猴子每天可以免费发10颗栗子，所以按照1香蕉=10栗子的比例，猴戊拿着他开的香蕉条子，一下换了一千多颗栗子。在猴山的另外一边，猴乙丙丁戊也在重复着类似的动作，一时间，猴山处，香蕉仿佛被神化了一般，既能增加记忆力，又能开发智力，既能壮阳，又能滋阴，简直神乎其神。
  由于五只猴对香蕉的大力宣传，掀起了猴山一股“香蕉换栗子”的浪潮，猴山里许多的猴子也开始来找我兑换香蕉券——有些打算趁机屯一把，有些则打算自己吃——当然，他们是用栗子来换的。一时间猴山的栗子都快被买空了，栗子大大涨价，现在1支香蕉，最多只能换5颗栗子了。现在我统计一下笼子五猴手里的栗子情况猴甲：持有栗子1000，香蕉票0猴乙：持有栗子900，香蕉票0猴丙：持有栗子900，香蕉票0猴丁：持有栗子800，香蕉票10猴戊，持有栗子1100，香蕉票0为啥猴戊手里的最多？因为猴戊从表弟那儿收购按了友情价，额外附赠了一些。猴山的猴子们不知道的是，此时囤积了大量栗子的猴甲乙丙丁戊们心理面已经准备开香槟了。此时，猴戊一声令下，五只猴子立马分散到猴山的各个角落——开始抛售持有的栗子了。猴山的猴子们傻眼了。随着五只猴子的抛售，栗子的价格大跌，此时1支香蕉能换20颗栗子了。当我这儿的五只猴子把栗子都换走之后，他们都大赚了一笔。猴甲：持有栗子0，香蕉票150猴乙：持有栗子0，香蕉票135猴丙：持有栗子0，香蕉票135猴丁：持有栗子0，香蕉票130猴戊：持有栗子0，香蕉票165五只猴子心满意足，这一波横扫猴山，真的让他们每只猴都血赚了一笔。猴山的猴子就欲哭无泪了，个别栗子多的猴存量被掏空了，个别猴没吃过香蕉，却背上了香蕉债……猴山的猴子们愤怒了，特别是猴戊的表弟，冲过来就要打猴戊，一套无影脚就要朝猴戊脸上甩过去时，冷不丁被从旁边闪出来的猴儿撞飞了，定睛一看，嘿，居然是猴丁——看来半只香蕉雇来的保镖，在关键时刻还真能救命！眼瞅着五只猴就要被猴山的猴子们围攻了，我赶紧进来救场。五只猴排着队，拿着香蕉的号码牌老老实实跟着我回笼去了。
  “发财咯！”一回到笼子里，猴丁便喜不自胜地跳了起来。“瞧你这嘚瑟样，是没见过香蕉吗？”猴甲揶揄到“这么说来，咱的债务不但还清了，还攒了几十支蕉，后面一段时间咱要好过很多啦”猴乙感叹道。“谁说你们债务还清了的？”我生气的说道“已经跟你们说了，不能打架，现在你们每只猴各被增加10支蕉的债务！”话音刚落，猴丁就崩溃了“我好不容易赚了几只香蕉，现在又赔进去了！”猴丙一听，抱怨道：“都说了，咱赚到蕉就该低调点，现在你们看，瞎嘚瑟了吧。”猴戊拍了拍猴丁的肩膀道：“兄弟，刚才多亏了你，你那多出来的1.5蕉的债，我帮你扛了”，猴丁感动的热泪盈眶。现在来梳理一下五只猴的债务情况猴甲：负债130.5支，香蕉票150猴乙：负债130支，香蕉票135猴丙：负债131支，香蕉票135猴丁：负债130支，香蕉票130猴戊：负债132支，香蕉票165猴戊摸了摸猴脑袋，慢条斯理地说：“咱这债务的事儿先搁一边，我倒觉得咱还能再赚一笔，你看这蕉已经借出来了，早还晚还利息都一样，不如咱再去捞一笔。”猴丁立马兴奋起来“戊哥！我这次绝对控制好情绪，再带我一次吧！”猴乙、猴丙也纷纷表示赞同，只有猴甲觉得自己赚够了，就不去了。四只猴子在去到西猴山之前，在猴戊的要求下，统统把毛发梳得锃亮，三七分、大背头啥的还整了个发型，屁股整的通红，手脚都洗的干干净净，一副“社会猴”的样子。猴戊还用它的香蕉票兑了那剩下的1.5支现货香蕉出来，也不知道它拿来有啥用。在我的首肯下，猴戊带着猴乙他们去到了动物园的另一座猴山——西猴山，这西猴山住的都是狒狒，数量得有几百只。四只猴刚走进西猴山，就被狒狒首领拦住了。
  首领瞪着眼前的不速之客，问道：“你们几个看着不像狒狒，到西猴山来干嘛？”猴乙站了出来，跟首领说道：“这位大哥，小弟一行来此宝地，是为了给西猴山所有的狒狒们一个发财的好机会，机会就在眼前了，咱要不要？”想不到，狒狒首领断然拒绝了“对不起，我们狒狒跟你们猴不一样，我们对发财不感兴趣”“对美食，也不感兴趣吗？”猴群中传来猴戊富有磁性和说服力的声音，几只猴子让开了一条道。“美食？什么美食？”狒狒首领似乎有了点兴趣“请看”，猴戊从背后拿出一包用叶子和草绳精细包裹好的东西，递给狒狒首领。狒狒首领打开一看——居然是半支香蕉“这什么破水果？居然还不是完整的”它生气地一把将叶子丢开，就要把香蕉连皮吞下。“这么高雅的珍品，居然这样吃？？？！！！”猴戊故做一副惊讶状。首领好歹是首领，没见过大场面是不可能的，被猴看成土包子更是万万不可能的，连忙把送到嘴边的带皮香蕉放在叶子上。只见猴戊不知从哪里掏出一把塑料小刀，先剥去香蕉已经氧化了一半多的外皮，再把香蕉仔细切成片，然后把包裹着香蕉的叶子，似餐巾一般围在了狒狒首领的脖子上，又掏出一把塑料小叉子，小心地递给首领——我就说上次中秋节有个冒失鬼丢了一整盒月饼进猴笼，怎么我清理时到处也找不到配送的塑料小刀叉了，原来在这！！！“您请慢用”首领疑惑地拿起叉子吃了一片。“这珍贵的食材，一定要细嚼慢咽”“这还要你说？！当我没吃过果子吗”首领慢慢咀嚼起来。
  “怎么样？是不是觉得这果子温润甜美、入口即化，好似阳春三月猴山顶上融化的雪一般，让你从喉头刺激到胃里？”猴戊不失时宜地问道。“首领？怎么样？好不好吃？”其他狒狒纷纷流下了幸福的口水。“恩，确实不错，软滑不腻，酸甜可口”首领说着，又尝了一片，嚼了嚼，补充道“最主要的是这东西，不塞牙缝！”。“不塞牙缝！！！！”听到这四个字，狒狒们两眼冒光。猴乙丙丁面面相觑——不塞牙缝怎么就这么好了？狒狒首领把剩下的几片香蕉分给几位长老，长老们吃了都觉得美味极了。“说吧，这东西怎么卖？我再买它几十支。”猴戊笑了笑“首领大哥，咱今天排除万难前来，难道只是为了跟您谈几十支香蕉的生意吗？”“胃口不小呢，咱狒狒吃水果，也就图个甜，要是填肚子，都不够我们吃一顿。”“那咱平时吃啥饱肚子？”“活鸡”猴戊有点懵圈，这个剧本好像有点不太对劲。“啥？你们居然有肉吃？”“那当然，我们狒狒是杂食动物，主要吃禽类，你来之前都不做市场调研的吗？”猴乙丙丁们终于知道不塞牙有多好了。“那咱平时如何获取、保存活鸡呢？”“活鸡票”以为狒狒长老说。“两只狒狒每天一只活鸡，活着的鸡吃不了，咱就存着，饲养员给咱发存鸡的票据，帮咱把鸡养起来，每只鸡2天得一颗鸡蛋作为利息。有时候咱开宴会，鸡不够了，还可以问饲养员借，借一只鸡每天收2个鸡蛋做利息”“有时我们光吃鸡蛋也能饱”首领补充道。
  猴子们目瞪口呆，同在一个动物园，为何狒狒条件如此优越……听到这，我也目瞪口呆，西猴山饲养员这娃，真是太厉害了。西猴山饲养员每天发半只鸡给狒狒，每2天再给一个鸡蛋做利息，其实就是打了狒狒们餐标的主意。这家伙净买母鸡，母鸡每天能下一颗蛋，也就是说每只鸡这家伙每两天能赚1颗蛋。有些蛋拿去发，有些蛋拿去孵，孵出公鸡了就当成肉鸡发给狒狒，孵出母鸡了就抓去下蛋，然后鼓励狒狒们多存鸡，靠鸡蛋度日，渐渐地他甚至都不需要再买鸡了。可猴戊觉得，自己仿佛到了一片充满了食物的大海里。“首领大人，这香蕉除了不塞牙缝，还能清洁牙齿”猴戊顿了顿“对了，清洁牙齿，就等于能避免口腔溃疡”。口腔溃疡几个字重重地砸在了狒狒们身上，长期吃肉塞牙缝的狒狒们都多多少少有口腔溃疡。“哼，我怎么知道你没在骗我？你告诉我，你到底打算卖什么？”听到这句话，猴戊退后了两步，呼唤来嘴皮子最厉害的猴乙，让它开始了它的表演 “首领大哥，您看，咱这香蕉是个好东西吧？”首领点点头。猴乙不知道从哪里捡了根小棍子，在土地上比划着——“这是鸡，这是蕉，蕉虽好，价也高！”首领听得一愣一愣的“香蕉好吃，每天一支，香甜可口，鲜嫩多汁”猴乙差点唱了出来。“也就是说，两支香蕉等于一支活鸡”狒狒首领想了想，好像也是“但是你们会种香蕉？”。
  “你们狒狒可能不知道，但是我们猴子对香蕉实在太了解了”猴乙顿了顿“在咱猴子云集的东猴山，有一片广阔的土地，勤劳的猴子们用自己的双爪开垦……”“打住打住！”首领听得一头雾水“说重点”“猴子”“种香蕉”“吃不完”猴乙连说带比划，终于把香蕉的故事讲完了。首领似乎明白了“你的意思是，你们猴子莫名其妙地学会种一种一星期就能结果的香蕉之后，种了一大堆，想跟我们换？”“不不不！”猴乙发现首领还是没明白，急得抓耳挠腮。“首领您误会了”猴戊也看不下去了。“香蕉树结一次果，就凋了，想继续吃，就得继续种。”“所以你们倒是种啊，”“所以咱不是来谈合作吗”猴戊终于把目的挑明了。“合作？什么合作？”首领疑惑到，“难道你想训练鸡帮你们种香蕉？”“您想想，如果您把平时用不上的活鸡票兑成香蕉票，然后投资给咱种香蕉，您出种咱出力，等蕉一熟，咱就按比例分，是不是美滋滋？”“可是活鸡还能生蛋，香蕉能生什么玩意儿”猴戊露出了难为情的样子“香蕉在结果之前当然啥都不能生了，但是咱回报高啊，您投资10只鸡，只用1个星期能换30支香蕉，您说是不是挺划算的。”狒狒召集来狒狒长老们，讨论了一下，都觉得是时候换换口味了，于是就同意了猴子们的提议。我惊呆了，照这样算，猴子们甚至都不用种香蕉就能稳赚一把。
